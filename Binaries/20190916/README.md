
## DB4O Object Manager GPL Binaries 8.1.2

### Java ObjectManager
```
Download dist.zip
cd dist
java -jar ObjectManager.jar
```


### .NET ObjectManager
```
Download Release.zip
cd Release
mono ObjectManager.exe
```
